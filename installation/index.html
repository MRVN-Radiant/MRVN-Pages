Running radiant for the first time

When running radiant for the first time it's important to know what the options in the popup below mean.

  1. Select the game - The game you want to make a map for. Each game has its own resources folder.

    You can change this in Edit>Preferences>Global>Game

  2. Show Global Preferences - Enable / Disable whether this window should display when you launch radiant.

    You can change this in Edit>Preferences>Global>Game
